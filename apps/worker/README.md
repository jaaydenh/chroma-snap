# @chroma-snap/worker

PNG diff worker and local baseline processor. It reads normalized manifests, compares screenshots to canonical base-branch baselines, classifies changed/new/deleted/errored/unchanged snapshots, writes diff images, persists comparison reports, records per-snapshot comparison errors without aborting the whole build, emits JSON-line metrics, can seed initial base-branch baselines for onboarding, and can reconcile approved PR changes on base-branch confirmation with `--reconcile-approved`. Milestone 7 also exports `createCleanupJobHandler()` for durable cleanup queue jobs that call the API cleanup endpoint.
